Privacy Policy

Introduction

This document refers to two groups of responsible persons

  • Data managers - the persons responsible for managing the data entered into the site and the general running of the service
  • Product owners - the persons responsible for developing and supporting the services provided to the data managers

Data managers and contact details can be seen on the About page of the site.

Data collected

Courage Scores will collect the following information about individuals:

  • Individual Name (Name and Given Name)
  • Team Name
  • Email Address (only for users who login)

The data will be used as follows:

Individual Name

The Name name will only be visible via the data API or wherever audit information is made visible on the website. Data audit information refers to details of who created, edited or deleted an item of data.

A players' name will be entered by data managers who record which individuals make up each team in the league. This data is entered and maintained by data managers. Historical data will be retained so that historical data can be reviewed.

Individuals have the right for their data to be updated or obfuscated at any point, by a request to the data managers.

Team Name

The name of each team will be recorded to represent the name of each team playing in each season.

The data is entered and maintained by data managers and can be updated if incorrect by request.

Email Address

The email address will never be exposed via the public version of the site. It is visible only by data managers of the site.

The email address is used to associate users with players in a team to allow for additional functionality when the individual logs in.

The email address is captured as part of the login process. The email address can be entered by the data managers to link users with players.

Any email address can be updated or removed from players by the data managers upon request.

Right to be Forgotten

Individual details can be removed from the site by one of the following options:

  • A request to the data managers, who can obfuscate any player from the site
  • A request to the product owner, via the data managers, who can remove any login information from background databases.

Data Security

Data will be held securely within the background systems provided to host the website.

Any issues with data security can be reported to the data owners by creating an issue here
